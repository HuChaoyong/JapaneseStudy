  • 回顾名词的接法

    动 and 名　　　　=>　原型　|　た + 名词.
    ①adj　and 名　　=>　①adj + 名， eg:　可愛い猫
    ②adj　and 名　　=>　②adj な 名，eg:　綺麗な猫
    名 and 名　　　　=>　名の名，　　eg: 日本語の本.

    【一堆修饰词】 + 猫，　　　该怎么写？
    【一堆修饰词】的简体形　+　猫。
    eg: これは資料です，明日会議で使います。
    　　これは【明日会議で使う】資料です.

    eg: 飛行機は中国航空です，私が明日乗ります。
    　【私が明日乗る】飛行機は中国航空です。

    これは葉子先生が日本の友達にもらった目薬です.

语法.

  • 如果我要用一个句子来放在这里【粉色的部分】. 那么我就要用.

    【句子简体】 + の　　=>　　が好き.
    形式体言.
    体言: 不会随时态变化的词.(数，量，代，名)
    用言: 形，动
    eg: コーラを飲む　の　が好き.
    我喜欢喝可乐.

  • の和こと 辨析.(from P15)

    の句子， 作为主语时， 一般不能和こと换.
    の句子, 做宾语时，可以和 こと互换.
    如果谓语动词是 “知らせる，伝えます、話す”等表示语言行为的动词时， 要用こと而不能用の.
    如果谓语动词是 "見る、聞く、"等表示感知的动词或者，“手伝います、やめます、防ぎます”等表示直接作用于动作对象的动词时，要用の.

语法.(主要讲授受关系)

  • 基本授受关系:

    AはBに花をあげる。 A把花给B.
    BはAに花をもらう。B收到来自A的花.
    Bは私に花をくれる。B给了我花.

  • 同样类比，把动词变成て形，完全如此，一样适用.

    〜てあげる
    〜てもらう
    〜てくれる

  • demo1

    私があなたに【荷物を持って】あげる。我给你【拿东西】.
    私は先輩に【荷物を持って】もらう。　　我收到前辈【给我拿行李这个】恩惠.
    先輩が私に【荷物を持って】くれる。　　前辈给我拿行李.
    一般在描述时，都是用　　私は、私が
    而在表示感谢时，一般是用
    〜てくれてありがどう。感谢〜

  • demo2

    明日学生にレポートを書いてもらう。(表命令，安排)
    我明天要让学生写一份报告.

  • demo3

    在使用　　私に〜てくれる。中.
    【私に】不是绝对的，有时候会是和我一方的人.
    比如感谢别人辅导我的妹妹时,此时，我妹妹就是和我一方的，对于外人来说.
    妹に宿題をやってくれてありがとう.
    谢谢(你)给我妹妹做作业.

  • demo4

    在使用这个【〜てあげる】句型时，有种，我施恩与你的感觉.
    适合对平辈，同级，下级使用
    近所をあんないしてあげる。
    而，对上级该这么用.
    近所を案内しましょうか。

语法 (动词意志形)

  • 变形规则:　(参考变形页面规则)

  • 意志形的作用:

    1.表明意志，要做〜，想做〜，决心〜，也可以表示邀请
    寝ろ！(表决心，要睡觉了.)
    一緒に飲みにいこうよ！(表邀请, 一起去喝一杯吧)
    2.猜测，推测，(以后学)

  • 意志形 + と思います。

    ①更委婉,
    ②用了　と思います　就只能表现为我的意志，去掉了“邀请”那层的意思.
    明日病院へ行こうと思います。

  • と思う

    第一人称:　〜ようと思う.(突然做的觉得， 坚定)
    第二人称:　〜ようと思っている.
    第三人称:　〜ようと思っている.(意志，持续了很长时间。 | 不怎么坚定)
    よう表示动词的意志形

  • 辨析:

    这2个词，都表示原因，理由
    から　　冲，上头，情绪化. ので　　客观，高冷(工作中，商务中). 陈述客观事件.
    所以，用から可以接意志形，而用ので，相对来说，不怎么合适.

  • 辨析:

    そろそろ　　接段时间内要去做的（移动性动作);
    そろそろ帰ろよ!　　差不多要回去了.
    もうすぐ　　快到某个时期(时间点)。
    もうすぐ夏休みだね！快到暑假了呀~

  • んです.

    A:明天安排大伙儿出去玩.
    B:どこへ行くんですか.(这里的んです表示进一步询问)

  • 〜ようとする　　正要做〜、眼看就要〜

    お風呂に入ろとしていたところ，電話がかかってきた.
    刚要洗澡的时候，电话来了.

  • 自动词 + ようとする　　显得很文艺

    夏休み終わろとしている。眼看暑假所剩无几.
    私の青春がおわろとしている。我的青春眼看就要逝去.

  • 〜でいっぱいです　　〜中充满了 ...

    私の胸に切ない気持ちでいっぱいです。　我心中满怀感伤.
    私の心に貴方のことでいっぱいです。我心中装满了你.
